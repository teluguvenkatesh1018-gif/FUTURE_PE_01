# Task 1 – AI Website Copy Generator

## Business Chosen
MIDNIGHT LOVE Café  
Location: Hyderabad

MIDNIGHT LOVE is a late-night café designed for students, couples, friends, and night owls who want a cozy place to relax, drink chai, and enjoy conversations.

---

## Objective
The goal of this task was to design a structured prompt that can generate high-quality website copy for a local business using AI.

The generated content should include:
- Homepage headline
- Subheadline
- Introduction
- Services section
- Call-to-action

---

## Prompt Logic
The prompt was designed to guide the AI by specifying:

- Business type
- Target audience
- Tone of communication
- Unique selling point
- Required website sections

This ensures the AI generates structured and relevant website content rather than generic text.

---

## Tool Used
ChatGPT

---

## Outcome
Using the designed prompt system, AI generated a complete website copy set for the café including homepage content, services, and a call-to-action section.

This framework can be reused to generate website copy for other local businesses as well.
