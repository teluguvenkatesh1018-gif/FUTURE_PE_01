# Prompt Engineering Internship – Future Interns

This repository contains the tasks completed as part of the **Prompt Engineering Internship Program at Future Interns**.

The project demonstrates how AI and prompt engineering can be used to create marketing content, website copy, and SEO blog systems for real businesses.

---

# Business Case Study

Business Name: **MIDNIGHT LOVE Café**  
Location: **Hyderabad**

MIDNIGHT LOVE is a late-night café designed for students, couples, friends, and night owls looking for a cozy place to relax, enjoy chai, and have meaningful conversations.

---

# Tasks Completed

## Task 1 – AI Website Copy Generator
Created structured prompts to generate conversion-focused website copy for the café including:

- Homepage headline
- Subheadline
- Introduction section
- Services / menu highlights
- Call-to-action section

---

## Task 2 – AI UGC Ad Script Generator
Designed prompts to generate high-converting UGC-style ad scripts for social media platforms like Instagram Reels.

The output includes:

- Multiple scroll-stopping hooks
- UGC-style ad scripts
- Call-to-action lines

---

## Task 3 – AI SEO Blog & Content Cluster Generator
Developed a prompt framework for generating SEO-focused blog content for the café website.

The SEO content pack includes:

- 1 pillar blog
- 3 supporting blogs
- Local SEO optimization for Hyderabad

This project demonstrates how prompt engineering can be used to build scalable content systems for:
